## Installation

### From VS Code Marketplace
1. Open VS Code
2. Press `Ctrl+P` (or `Cmd+P` on Mac)
3. Type `ext install digitalforge.cb42448e`
4. Press Enter

### From VSIX file
1. Download the `.vsix` file
2. In VS Code: `Extensions > ··· > Install from VSIX`
3. Select the downloaded file

## Usage

Press `Ctrl+Shift+P` and search for `Logistics & Shipping API Client and Request Builder VS Code Extension` to see all available commands.

**Keyboard shortcut:** `Ctrl+Shift+Alt+A` (or `Cmd+Shift+Alt+A` on Mac)

## Settings

| Setting | Default | Description |
|---------|---------|-------------|
| `cb42448e.enabled` | `true` | Enable/disable the extension |
| `cb42448e.autoRun` | `false` | Auto-run on file open |
| `cb42448e.logLevel` | `"info"` | Log verbosity |
